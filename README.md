# Georg Stage vagtplanlægger

Dette er et hjælpeprogram til vagtplanlægning ombord på Georg Stage.

- Holmen
- Print til pdf med dato-interval
- EASTER EGG: Play 6 bakke lyd når man trykker på 6. bakke
